# Codebook for the Getting-and-Cleaning-Data Course Project
================

This file continas the list of variables names and their descriptions for the tidy data output of the run_analysis.R script. 

## Introduction

..........Project for the Coursera course entitled "Getting and Cleaning Data" using Samsung Galaxy S accelerometer and gyroscope data. **fix**

	Activity performed by subject during measurement; 6 levels
		WALKING
		WALKING_UPSTAIRS
		WALKING_DOWNSTAIRS
		SITTING
		STANDING
		LAYING
